
<p align="center">A web-based To Do app </p>

### Installation

```bash
git clone git@github.com:anazmy14/To-Do.git
cd To-Do
npm i
```

#### Run app. 


```bash
npm start
```

Go to `localhost:3000` to view the app.

## Testing

Run tests.

```bash
# In one window, run the application
npm start 

# In another window, run the tests
npm run test
```
